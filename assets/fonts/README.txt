Reserved for self-hosted font files.

The site currently loads "Orbitron" and "Rajdhani" from Google Fonts (see the
<link> tag in index.html's <head>). If you'd rather self-host them:

1. Download the .woff2 files for both families.
2. Drop them in this folder.
3. Add @font-face rules at the top of style.css pointing to them.
4. Remove the Google Fonts <link> tag from index.html.
