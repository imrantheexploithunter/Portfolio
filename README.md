# ImranTheExploitHunter
Imran Chowdhury is a Cyber Security Researcher and the founder of ImranTheExploitHunter, a research initiative dedicated to exploring offensive security techniques, threat intelligence, and open-source tooling for the security community.

His work spans Red Team operations, Active Directory security, malware analysis and detection engineering, with an emphasis on turning hands-on lab research into practical, publicly shared knowledge through write-ups, tools and video walkthroughs.

Beyond hands-on testing, Imran maintains a set of open-source security projects on GitHub and publishes research breakdowns on Medium, aiming to make offensive security concepts more accessible to practitioners entering the field.
